Sobre o Projeto

Nosso site é dedicado a fornecer informações atualizadas sobre a COVID-19 no Brasil e no mundo, com a utilização de gráficos como:
Gráfico mapa, Gráfico pizza e Tabela. O objetivo do nosso site é passar informações para o usuário mantendo-o informado sobre o Corona Vírus e como está o andamento dos casos atualmente.
Esse site foi produzido por alunos do SENAI, um trabalho proposto pelos professores Sérgio, Pafúncio e Zuleika. A plataforma foi desenvolvida com uso do Visual Studio Code com auxilio da ferramenta API para a aquisição de informações, pois nos possibilita ter os dados em tempo real e atualizados. Para a criação de cada gráfico, utilizamos a galeira de gráficos disponível no Google Charts.
